# 桌面救星 / Desktop Savior

通用、谨慎、可回溯的个人文件整理与数字档案助手。

A context-aware, safe, and reversible organizer for mixed personal and work files.

---

## 这是什么 / What Is This

**桌面救星** 用于整理混杂在桌面、下载、文档或指定目录中的个人数字资产。它不假设内容只能是工作文件或生活资料：当前工作、以往工作归档、学习记录、照片视频、创作项目、兴趣收藏、工具资源与敏感资料都可以共存。

它会先进行只读盘点，再根据“使用状态 + 项目/事件/主题语境”提出整理计划；只有获得确认后才会分批移动，并保留日志、索引、计划快照和撤回路径。

## 核心能力 / Capabilities

| 能力 | 说明 |
|------|------|
| 首次盘点 | 只读扫描目标目录，识别活跃、沉淀、记忆、资源、敏感与待确认资料 |
| 通用分类 | 支持按状态、项目经历、事件时间、主题或混合结构整理，不预设工作或生活优先 |
| 增量整理 | 复用已确认的规则，只处理新增项目和冲突项 |
| 查找解释 | 根据文件名、原路径、项目/事件标签、归档批次找回资料，并解释移动原因 |
| 安全撤回 | 按运行编号生成反向计划，小批量恢复到原位置 |
| 高保护模式 | 对创作版本、媒体原片、近期文件、敏感资料默认仅索引或待确认 |

## 安全底线 / Safety

- 不删除、不清空废纸篓、不覆盖、不自动重命名。
- 不自动去重、不按“旧文件”清理、不拆分既有文件夹。
- 不读取敏感正文，不记录证件号码、账号、密钥或私人内容。
- 首轮只读；移动前给出具体路径计划；每批最多 10 项并逐项验证。
- 每次移动都有日志和索引，支持按 `run_id` 生成撤回计划。

## 触发方式 / How To Use

- “用桌面救星盘点一下我的桌面，先不要移动。”
- “整理下载目录，当前工作和旧资料要区分开，但先给我方案。”
- “按上次规则整理新增文件，创作资料不要自动移动。”
- “帮我找去年某个项目的归档文件。”
- “撤回桌面救星上周那次整理。”
